The .vcm file is a Javelin UML class diagram/design file. To open the design and view and or edit the class diagram simply download the Javelin Modeler or Javelin Modeler+Coder from:

	http://stepaheadsoftware.com/products/javelin.htm

Javelin is a lightweight unobtrusive visual object development enviroment for Java. It is a Windows app that is also compatible with Wine on Linux so it can run on both Windows and Linux.
